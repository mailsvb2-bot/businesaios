# Ads Connector Layer — правила и безопасность

## Зачем отдельный слой

Подключение рекламных кабинетов без внутренней пользы и guardrails = финансовый риск.

Поэтому Ads Connector Layer в BusinesAIOS устроен так:

- **Есть общий интерфейс** (`interfaces/ads/base.py`): connect/exchange_code, inventory, metrics, gated write.
- **По умолчанию используется read-only**: коннекторы содержат только безопасные read-методы; write-заглушки явно запрещают изменения.
- Любые write-операции допускаются только при включённых **entitlements + budget guardrails** и при наличии **audit trail**.

## Минимальные предпосылки перед включением Ads

1) Стабильная воронка внутри (lead → purchase)

2) Офферы/цены/скрипты живут в каталоге и изменяются детерминированно

3) Включены stop-loss правила и дневные лимиты

4) Есть attribution (фиксируется при создании лида и приклеивается к покупке)

Каноничный модуль: `core/growth/attribution.py`.

## Контракт ответственности

Слой коннекторов **не является автопилотом сам по себе**.

Он предоставляет:
- чтение кампаний/метрик
- единый нормализованный формат метрик
- поверхность для write-операций (но они заблокированы по умолчанию)

Автопилот (если включается) должен жить отдельным сервисом и быть обязан:
- проходить через **BudgetGuardrails** (`core/growth/budget_guardrails.py`)
- оставлять **audit trail** в event stream
- иметь kill-switch и лимиты

## Audit mode

Audit-mode = любые попытки применения изменений (даже человеком) обязаны:
- фиксировать intent/patch
- фиксировать решения guardrails (ok/blocked + reason)
- фиксировать результат платформы (success/fail)

## События (каноничный минимум)

Attribution:
- `growth_lead_attribution_captured`
- `growth_purchase_attributed`

Safety:
- `ads_guardrails_block`

Реклама-метрики (если импортируются в поток событий):
- события импорта метрик должны быть явными и версионированными (не смешивать с продуктовой телеметрией)

## Guardrails

- дневной лимит бюджета (total/per-platform/per-campaign)
- max % увеличения бюджета за шаг
- запрет креативов по умолчанию
- окно времени для изменений (UTC)
