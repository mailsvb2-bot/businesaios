# CANON EXCEPTION REGISTRY REMEDIATION V1

Цель:
оформлять исключения канонично, а не прятать нарушения.

## Типовые исправления
- добавить entry в `docs/CANON_EXCEPTION_REGISTRY_DATA_V1.yaml`
- назначить owner
- добавить expires_on
- указать точные `paths`
- закрыть или продлить просроченное исключение явно

## Главный смысл

Исключение — это не способ обойти конституцию.
Это способ:
- зафиксировать долг
- ограничить его
- сделать его видимым
- не дать ему стать нормой
